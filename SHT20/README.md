## SHT20
El SHT20 es un sensor que mide la temperatura y la humedad y envia la información a traves de I2C.

<img src="https://cdn-shop.adafruit.com/1200x900/4099-09.jpg" width="400" height="300" />

Dispone de 4 cables
* Rojo      ➡️  VCC (+5V)
* Negro     ➡️  GND
* Amarillo  ➡️  SDA (DATOS)
* Blanco    ➡️  SCL (CLOCK)
